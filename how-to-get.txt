How to get turbo nibby.

1. Open https://turbowarp.org/

2. Make a new tab and open https://github.com/TheRealNibby/Turbo-Nibby-For-Turbowarp

3. Select turbonibby.js in github

4. Copy all the code from turbonibby.js

5. Reopen the https://turbowarp.org/ tab

6. At the top click See Inside

7. Click the red extension button in the bottom left

8. Tap the Turbowarp button

9. Select custom extension

10. Click text

11. Paste the copied code into the box.

12. Tap load

All done, Enjoy the extension
